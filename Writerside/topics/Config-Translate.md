# 配置机器翻译

DashPlayer 目前支持使用：

-   腾讯云翻译字幕
-   有道云翻译单词（鼠标**放置**在视频下方字幕行的单词上）
-   点击单词播放发音

要启用翻译功能，需要申请[腾讯云](https://console.cloud.tencent.com/cam/capi)和[有道云](https://ai.youdao.com/console/#/service-singleton/text-translation)的密钥，并在设置中填入
